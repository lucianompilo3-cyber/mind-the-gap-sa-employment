# Mind the Gap: Analysing Graduate Employability in South Africa

## Project Overview
This project analyses provincial unemployment trends in South Africa 
using Stats SA QLFS Q1 2026 data.

## Key Findings
- Eastern Cape has the highest unemployment at 44.6%
- Western Cape has the lowest at 19.6%
- National average stands at 33.4%
- Eastern Cape worsened the most year-on-year (+5.3 percentage points)

## Tools Used
- Python (pandas, matplotlib)
- Google Colab
- Microsoft Power BI

## Data Source
Statistics South Africa - Quarterly Labour Force Survey Q1 2026 (P0211)
